# current_datetime ツール設計書

## 1. 背景と目的
現在、LLM（生成AIエージェント）が現在の日時を取得しようとする際、`bash` ツールを経由して `powershell` や `node` のコマンドを実行しようとします。しかし、これは実行確認ポインターを出したり、コマンドのエラーで失敗することが多く、非効率的です。
そのため、生成AIが直接現在の日時を取得できる専用ツール `current_datetime` を追加し、日時取得を安全かつ確実に行えるようにします。

## 2. ツール仕様
- **ツール名**: `current_datetime`
- **説明**: 現在の日時（日時情報）を取得します。引数は不要です。
- **入力パラメータ**: なし
- **出力結果**:
  - ISO 8601形式の日時文字列 (例: `2026-03-07T14:29:26.000Z`)
  - ローカルタイムゾーン形式の日時文字列 (例: `2026/3/7 23:29:26`)
  - タイムゾーンオフセット情報

## 3. 実装方針
1. `src/tools/definitions/current-datetime.ts` を作成する。
   他のツールと同様に `ToolHandler` インターフェースを実装する。
2. `src/index.ts` にて、作成したツールをインポートし、`toolRegistry.register(...)` を用いて登録する。
3. `TODO.md` を作成・更新し、実装ステップを管理する。各実装ステップ完了時にGitコミットおよびプッシュを行う。
