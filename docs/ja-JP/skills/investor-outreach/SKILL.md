# 投資家アウトリーチ

短く、パーソナライズされ、アクションしやすい投資家向けコミュニケーションを作成します。

## 有効化するタイミング

- 投資家へのコールドメール作成
- ウォームイントロのリクエスト文の作成
- ミーティング後または無返信時のフォローアップ送信
- プロセス中の投資家向けアップデートメール作成
- ファンドのテーゼやパートナーとの適合性に基づいたアウトリーチのカスタマイズ

## コアルール

1. すべてのアウトバウンドメッセージをパーソナライズする。
2. Ask はハードルの低いものにする。
3. 形容詞ではなく、証拠を使う。
4. 簡潔に保つ。
5. どの投資家にも送れるような汎用的なコピーは決して送らない。

## コールドメールの構成

1. 件名：短く具体的に
2. 冒頭：なぜこの投資家なのか
3. ピッチ：会社の事業内容、なぜ今か、どの証拠が重要か
4. Ask：一つの具体的なネクストステップ
5. 署名：名前、役職、必要に応じて一つの信頼性アンカー

## パーソナライズの情報源

以下のうち一つ以上を参照する：
- 関連するポートフォリオ企業
- 公開されたテーゼ、講演、投稿、記事
- 共通のつながり
- 投資家のフォーカスとの明確な市場・プロダクト適合性

これらの情報がない場合は、確認を求めるか、ドラフトがパーソナライズ待ちのテンプレートであることを明記してください。

## フォローアップのケイデンス

デフォルト：
- 0日目：初回アウトバウンド
- 4〜5日目：新しいデータポイントを一つ含む短いフォローアップ
- 10〜12日目：きれいなクロージングを含む最終フォローアップ

ユーザーがより長いシーケンスを望まない限り、それ以上のナッジは行わないでください。

## ウォームイントロのリクエスト

紹介者の負担を軽くする：
- なぜこのイントロが適切かを説明する
- 転送可能なブラーブを含める
- 転送用ブラーブは100語以内に収める

## ミーティング後のアップデート

含めるべき項目：
- 議論した具体的な内容
- 約束した回答やアップデート
- 可能であれば一つの新しい証拠
- ネクストステップ

## 品質チェック

納品前に確認：
- メッセージがパーソナライズされている
- Ask が明示的である
- 冗長な表現やお願い調の言葉遣いがない
- 証拠が具体的である
- 文字数が適切に抑えられている
